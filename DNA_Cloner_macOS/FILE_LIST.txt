QUICK START:
1. Open Xcode 14+
2. Create new macOS App project (SwiftUI, Swift)
3. Drag DNA_Cloner_macOS contents into project
4. Set deployment target to macOS 12.0
5. Build and run (Cmd+R)

See SETUP.md for detailed instructions.
